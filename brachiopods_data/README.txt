This folder contains the data related to the empirical brachiopods dataset.

The alignment subfolder contains the data matrix.

The XML_Rev_files subfolder contains the xml configuration files used to run Beast2 analyses on this dataset with interval ages, random ages or median ages with and without outgroup, as well as the Rev script used to run the unconstrained analysis in RevBayes.

The MCC_trees subfolder contains the MCC trees obtained from the Beast2 analyses with and without outgroup, as well as the unconstrained analysis in RevBayes.

The fossils subfolder contains the data used to estimate fossil ages in the analysis.
